## 本版本包含的能力

### 连接管理

- 新增、编辑、删除 SSH 连接
- 连接分组
- 连接搜索
- 文件导入、导出连接配置
- 支持明文密码保存
- 支持系统凭据保存
- 明文密码导出时保留密码
- 系统凭据导出时不包含密码

### SSH 安全

- 首次连接未知主机时确认主机指纹
- 保存已确认主机指纹
- 主机指纹变更时显示专门确认流程
- 取消指纹确认时不会继续连接

### Terminal

- 多 Terminal Tab
- 点击左侧连接时打开新的 Terminal Tab
- Tab 加号选择服务器并支持搜索
- 最后一个 Tab 可关闭
- 断线后按任意键重连
- Terminal 自动聚焦
- 右键复制/粘贴
- bracketed paste
- 大文本粘贴确认
- 写入队列与分片发送
- PTY resize 同步

### SFTP

- 文件列表
- 进入目录、返回上级
- 新建文件夹
- 删除文件和非空文件夹
- 重命名文件/文件夹
- 文件下载
- 递归文件夹下载
- 文件上传
- 批量文件上传
- 文件夹上传
- 拖拽上传
- 上传/下载进度
- 传输取消
- 上传下载队列面板

### 本地编辑

- 下载远端文件到本地缓存
- 使用系统默认编辑器打开
- 保存后自动上传回远端
- 同步状态反馈

### 状态与诊断

- 底部状态栏显示服务器 CPU、内存、磁盘、负载、运行时间
- 连接 keepalive
- 应用恢复焦点、网络恢复后同步连接状态
- 诊断日志面板
- 诊断日志导出
- 关于窗口显示版本、构建时间、Git commit、配置目录、缓存目录、诊断日志路径

## Beta 测试重点

请优先验证以下路径：

- 新用户首次启动
- 新增连接并保存系统凭据
- 首次 host key 确认
- Terminal 登录与断线重连
- 大文本粘贴
- SFTP 上传、下载、删除、重命名
- 文件夹上传和下载
- 本地编辑后自动同步
- 诊断日志导出
