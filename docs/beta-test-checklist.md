# LucidShell Beta 测试清单

本文档用于每次 Beta 发包前的本地验证。目标不是覆盖所有边界，而是确保安装包、核心 SSH/SFTP 工作流、凭据、诊断能力在真实使用路径上可用。

## 1. 构建前检查

- 确认工作区没有无关改动：`git status --short`
- 确认版本号符合预期：
  - `package.json`
  - `src-tauri/tauri.conf.json`
- 确认构建信息已生成：`src/shared/config/buildInfo.ts`
- 确认“关于 LucidShell”里显示正确的版本、构建时间和 Git commit
- 确认应用信息符合预期：
  - 应用名：LucidShell
  - Bundle Identifier：`app.lucidshell.dev`
  - 窗口默认居中启动
  - 自绘窗口栏可拖动、可最小化、最大化、关闭
- 确认图标资源存在：
  - `assets/brand/lucidshell-icon.svg`
  - `src-tauri/icons-lucidshell/32x32.png`
  - `src-tauri/icons-lucidshell/128x128.png`
  - `src-tauri/icons-lucidshell/128x128@2x.png`
  - `src-tauri/icons-lucidshell/icon.ico`
  - `src-tauri/icons-lucidshell/icon.icns`

## 2. 本地构建

- 前端构建：`npm run build`
- Rust 检查：`cargo check`，工作目录为 `src-tauri`
- Tauri 打包：`npm run tauri build`
- 确认安装包生成在 `src-tauri/target/release/bundle/`
- 安装包能正常安装、启动、卸载

## 3. 首次启动测试

- 清空或临时移走当前用户的 LucidShell 配置目录后启动
- 应用启动时不闪白、窗口居中
- 左侧连接栏没有异常默认选中状态
- 没有 Terminal Tab 时显示空状态和选择服务器入口
- 设置窗口可以打开，主题切换可用
- 诊断日志面板可以打开，空状态显示正常

## 4. 连接配置测试

- 新增密码连接
- 编辑连接
- 删除连接
- 新增分组
- 移动连接到分组
- 搜索连接
- 导出连接到用户选择的位置
- 从文件导入连接
- 明文密码保存时：
  - 编辑时可按眼睛按钮查看
  - 导出文件包含明文密码
  - UI 有明确提示
- 系统凭据保存时：
  - 保存后可以连接
  - 编辑时密码框显示占位符
  - 导出文件不包含系统凭据密码
  - 删除连接后不再保留对应凭据

## 5. Host Key 测试

- 首次连接未知主机时弹出指纹确认
- 确认后保存指纹并连接
- 再次连接同一主机不重复确认
- 指纹不匹配时显示专门的变更确认
- 取消确认后不应继续连接
- 确认更新后保存新指纹并连接

## 6. Terminal 测试

- 点击左侧连接打开新的 Terminal Tab
- 点击 Tab 加号弹出服务器选择 Dialog，并支持搜索
- 最后一个 Tab 可以关闭
- 切换 Tab 后光标自动聚焦到终端
- 右键有选中内容时复制
- 右键无选中内容时粘贴
- 粘贴多行命令时 bracketed paste 生效
- 粘贴大量文本时出现确认提示
- 大量粘贴不会阻塞其他 Terminal Tab 输入
- 调整窗口大小后 PTY 尺寸同步
- 断开连接后显示“按任意键重新连接”
- 网络恢复或应用重新获得焦点后连接状态能同步

## 7. SFTP 测试

- SFTP 初始目录可加载
- 双击文件夹进入目录时有加载反馈
- 返回上级目录
- 新建文件夹
- 重命名文件
- 重命名文件夹
- 删除文件
- 删除非空文件夹
- 下载文件
- 下载文件夹
- 上传单个文件
- 批量上传文件
- 拖拽上传文件到 SFTP 区域
- 上传文件夹并保留相对目录结构
- 上传/下载进度显示正确
- 上传/下载任务可取消
- 传输队列面板显示当前任务、已完成、待处理信息

## 8. 本地编辑测试

- 对远端文件执行本地编辑
- 文件下载到本地缓存目录
- 系统默认编辑器可以打开
- 保存本地文件后自动上传回远端
- 同步成功后 SFTP 刷新
- 同步失败时出现错误提示并写入诊断日志

## 9. 监控与状态栏测试

- 已连接服务器底部状态栏显示 CPU、内存、磁盘、负载、运行时间
- 监控命令失败时不影响 Terminal 输入
- 切换 Tab 后状态栏切换到对应服务器
- 断开连接后状态栏不显示过期的正常状态

## 10. 诊断日志测试

- SSH 连接失败会写入诊断日志
- Terminal 断开或写入失败会写入诊断日志
- SFTP 上传、下载、删除、重命名失败会写入诊断日志
- Keychain 保存失败会写入诊断日志
- 设置中的诊断面板能显示最近日志
- 可以清空诊断日志
- 可以导出诊断日志到用户选择的位置
- 导出的日志不应包含密码、私钥内容

## 11. 打包后安装验证

- 在当前开发机安装打包产物
- 在一台干净 Windows 用户环境安装打包产物
- 安装后启动无白屏、无权限报错
- 连接配置写入 app config 目录
- 本地编辑缓存写入 app cache 目录
- 诊断日志写入 app config 目录
- 卸载后应用主体被移除
